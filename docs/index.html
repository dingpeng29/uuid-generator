💡 UUID 使用场景

  • 数据库主键 — 分布式系统唯一ID，避免自增ID冲突
  • 文件名 — 生成唯一文件名，防止覆盖

  • 事务ID / 追踪ID — 分布式链路追踪
  • UUID v4 — 基于随机数，最常用
  • UUID v1 — 基于时间戳+MAC地址，包含时间信息
